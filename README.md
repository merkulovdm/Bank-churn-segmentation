# Bank-churn-segmentation
Customer churn analysis and client segmentation for a regional banking product.

# Bank Churn Segmentation Analysis

## Overview

This project analyzes customer churn behavior for a regional banking product.

The objective was to identify key churn drivers, validate behavioral hypotheses, and segment customers based on risk factors.

## Dataset

The dataset contains customer information including:
- credit score
- account balance
- number of products
- customer activity
- estimated salary
- churn indicator

The analysis focused on customers from several regional cities.

## Analysis Scope

The analysis included:
- exploratory data analysis
- churn factor analysis
- statistical hypothesis testing
- customer segmentation
- churn risk evaluation

# Key Findings

- Customers using only one product showed significantly higher churn rates
- Lower credit score was associated with increased churn probability
- Geographic differences had limited impact on customer retention

The highest-risk segment included customers with:
- high account balances
- multiple products
- elevated churn rates (~40%)

This indicated that high-value customers were not sufficiently retained despite strong product engagement.

# Segmentation Results

Three customer segments were identified based on:
- balance
- number of products
- age
- credit score

The analysis revealed distinct churn behaviors across customer groups and highlighted high-value clients with elevated retention risk.

# Recommendations

- Develop personalized retention strategies for high-value customers
- Improve cross-selling initiatives for low-engagement clients
- Monitor churn indicators continuously
- Use identified features for future churn prediction models

# Business Interpretation

The analysis suggests that high-balance customers may be more sensitive to service quality, pricing conditions, and competitor offers.

Retention strategies should therefore prioritize personalization and premium customer experience.
